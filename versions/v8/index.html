Spielregeln

🐍 Das Ziel: Klicke auf die Schlangenköpfe, damit sie sich aus dem Dschungel schlängeln. Entwirre das Chaos!

🚧 Die Regeln: Eine Schlange kann sich nur auf den Weg machen, wenn die gerade Linie von ihrem Kopf bis zum Spielfeldrand komplett frei von anderen Schlangenkörpern ist.

Features:

  • ❤️ Leben: Du hast eine begrenzte Anzahl an Leben pro Level. Klickst du eine blockierte Schlange an, verlierst du ein Leben. Bei 0 Leben startet das Level neu!
  • 🪨 Felsen: In höheren Leveln tauchen Felsen auf, um die herum du planen musst.
  • ♾️ Endlos lösbar: Jedes Level wird dynamisch und garantiert lösbar generiert. Es gibt 99 Schwierigkeitsgrade!
